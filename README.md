# GalwayLibrary
**Pair Programming Project**  

**Submitted in fulfillment of:**  
**Course Code:** 2526-CT5210 Modern Software Engineering  

**Partners:** Fintan Mitchell and Samuel Ukachukwu  


## Objective
Pair programming, SOLID principles, and Clean Architecture.  


## System Features
- Add a new book  
- Register a library member  
- Borrow a book  
- Return a book  
- View all available books  


## Project Rules
- No database is required (use ArrayLists).  
- No UML diagrams are required.  
- No unit testing is required.  
- Your code must follow the given folder structure.  


## Required Clean Architecture Folder Structure

```
galwaylibrary
 └── src
      ├── entities
      ├── usecases
      ├── adapters
      ├── frameworks
      └── main
```
